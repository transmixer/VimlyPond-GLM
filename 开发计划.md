# Vimlypond 开发计划

## ✅ 已完成功能（v0.10.0）

### 核心架构
- [x] Vim 风格模态编辑（普通模式/插入模式）
- [x] VexFlow 实时渲染
- [x] Zustand 状态管理
- [x] LocalStorage 持久化
- [x] 中英文双语界面

### 音符输入
- [x] 音名输入（a-g），智能八度推断
- [x] 休止符输入（r）
- [x] 时值修改（1, 2, 4, 8, 16）
- [x] 附点（.）
- [x] 升降号（+/-）
- [x] 八度变化（'/,）
- [x] 和弦输入（Shift+a-g）

### 导航操作
- [x] 小节内导航（h/l）
- [x] 谱表间导航（j/k）
- [x] 小节间导航（n/b）

### 乐谱结构
- [x] 添加谱表（o/O）
- [x] 切换谱号（t）
- [x] 调号支持（15大调 + 15小调）
- [x] 拍号支持（常用拍号）
- [x] 删除元素（x）
- [x] 撤销/重做（u/Ctrl+R）
- [x] 重复操作（.）

### 高级功能
- [x] 跨小节延音线
- [x] 流式推拉平衡系统
- [x] LilyPond 导出

## 💡 提议中功能

| 功能 | 为什么要做 | 技术方案 | 影响范围 | 测试要点 | 状态 |
|:---|:---|:---|:---|:---|:---|
| 播放试听 | 用户需要听到输入的音符，验证创作意图；这是记谱软件的基本功能 | 使用 Web Audio API，将 MIDI 音高转为频率；添加播放状态到 store；空格键播放当前小节 | 新增 `playback.ts` 模块，store 添加播放状态，编辑器添加播放按钮 | 输入音符后按空格播放，检查音高、时值正确 | ⏳ 待确认 |
| 连音输入 | 三连音、五连音等是常见节奏型，缺少此功能无法记谱大量作品 | 扩展 Note 类型添加 tuplet 字段；VexFlow 原生支持；LilyPond 用 `\tuplet` 语法 | Note 类型、时值计算、渲染器、导出模块 | 输入三连音，检查渲染和导出 | ⏳ 待确认 |
| 反复记号 | 乐谱必备功能，没有反复记号的乐谱不完整 | 在 Measure 添加 repeatStart/repeatEnd 标记；VexFlow setBegin/EndBarType | Measure 类型、渲染器、导出模块 | 添加反复记号，检查渲染和导出 | ⏳ 待确认 |
| 力度标记 | 让演奏更真实，支持 p、mp、mf、f 等表情记号 | 在 Note 添加 velocity 字段；渲染器显示力度文字；LilyPond 导出 `\p` `\f` 等 | Note 类型、渲染器、导出模块 | 添加力度，检查导出 LilyPond 含力度标记 | ⏳ 待确认 |
| 装饰音 | 倚音、颤音、波音等装饰音是古典音乐的重要组成部分 | 新增 Ornament 类型；VexFlow 支持多种装饰音；LilyPond 用 `\grace` 等语法 | 新增类型、渲染器、导出模块 | 添加倚音，检查渲染和导出 | ⏳ 待确认 |
| 多声部 | 一个谱表上同时显示多个声部，复调音乐必需 | Measure.elements 支持多声部数组；VexFlow 支持多声部渲染 | Measure 类型、渲染器、导航逻辑 | 输入两个声部，检查渲染正确 | ⏳ 待确认 |
| 鼠标定位 | 降低入门门槛，让不熟悉 Vim 的用户也能使用 | 监听 SVG 点击事件，根据坐标反算音符位置；设置光标位置 | 渲染器记录音符位置，store 添加点击处理 | 点击音符，光标跳转到正确位置 | ⏳ 待确认 |
| 宏录制 | Vim 核心特性，复杂记谱模式一键完成 | store 记录按键序列；播放宏时重放序列 | store 添加宏状态，编辑器添加录制/播放逻辑 | 录制一系列操作，播放宏重放 | ⏳ 待确认 |
| MIDI 导入 | 用户已有 MIDI 文件，想直接生成乐谱 | 使用 `midi-parser-js` 解析 MIDI；映射到现有 Note 数据结构 | 新增解析模块 | 导入简单 MIDI 文件，检查音符正确 | ⏳ 待确认 |
| PDF 导出 | 方便打印和分享，无需 LilyPond 编译环境 | 使用 `jspdf` + 渲染好的 SVG 直接转 PDF | 新增导出模块 | 点击导出，检查 PDF 文件 | ⏳ 待确认 |

## 🎯 功能优先级说明

**第一优先级 - 核心功能缺失**：
- 播放试听、连音输入、反复记号

**第二优先级 - 表现力增强**：
- 力度标记、装饰音、多声部

**第三优先级 - 用户体验**：
- 鼠标定位、宏录制

**第四优先级 - 互操作性**：
- MIDI 导入、PDF 导出
