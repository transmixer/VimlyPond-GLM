# Vimlypond 开发计划

## 项目目标

**VimlyPond = Vim 效率 + LilyPond 专业乐谱排版**

核心定位：通过 Vim 风格的高效键盘操作，完成专业级乐谱编辑，最终通过 LilyPond 生成印刷级乐谱。

用户工作流：
1. 键盘快速输入音符
2. 实时预览谱面效果
3. 导出 `.ly` 文件
4. 使用 LilyPond 编译生成 PDF/MIDI

## 🎯 开发原则

### 核心原则
1. **专注编辑器本体** - 完善乐谱谱面编辑的基本功能
2. **LilyPond 优先** - 所有功能必须支持 LilyPond 导出
3. **Vim 风格** - 保持键盘操作的高效性

### 不在范围内
- 播放试听 / MIDI 播放 - 用户通过 LilyPond 编译生成 MIDI
- MIDI 导入 - 非核心功能
- PDF 导出 - LilyPond 编译即可生成
